So each factor \(5^{3}\text{,}\) \(5^{0.1}\text{,}\) \(5^{0.04}\text{,}\) \(5^{0.001}\text{,}\) \(5^{0.0005}\text{,}\) \(5^{0.00009}\) is getting closer and closer to \(0\text{.}\) So then \(5^\pi\) should be really tiny, close to 0.”
